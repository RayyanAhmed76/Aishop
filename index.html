Submit to n8n

Upload documents and/or paste public URLs. Your browser sends JSON with base64 files; the API uploads each file to Vercel Blob, then sends n8n a JSON payload where each file is a public HTTPS link (files[].url), plus urls and metadata.

Documents

Word, PDF, plain text, PowerPoint, or CSV — multiple files allowed. The browser sends base64; n8n receives files[].url only (requires Blob env on the server).

    Links

    One URL per line, or separate with commas. Use publicly reachable URLs (not localhost) so n8n cloud can fetch them.
